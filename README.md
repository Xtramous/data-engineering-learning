# Data Engineering

Learning and projects for data engineering technologies including:
- Spark
- Airflow
- Kafka
- dbt
- Docker

## Directory Structure

- `spark/` - Apache Spark projects and notebooks
- `airflow/` - Airflow DAGs and workflows
- `kafka/` - Kafka streaming projects
- `dbt/` - dbt data transformation projects
- `docker/` - Docker configurations and containerization
